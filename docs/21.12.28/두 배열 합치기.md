### **<복습> 2. 두 배열 합치기**

Q. 오름차순으로 정렬이 된 두 배열이 주어지면 두 배열을 오름차순으로 합쳐 출력하는 프로그램 을 작성하세요.



**<풀이 방식>**
```java
import java.util.*;
class Main {
    public static ArrayList<Integer> solution (int n, int m, int[] a, int[] b) {
        ArrayList<Integer> answer = new ArrayList<>();
        int p1 = 0, p2 = 0;
        while (p1 < n && p2 < m) { // && 는 둘중에 하나만 거짓이 되도 거짓
            if (a[p1] < b[p2]) {
                answer.add(a[p1++]); // p1이 가리키는 값을 add 한 후 1이 증가한다.
            } else answer.add(b[p2++]);
        }
        // a 배열과 b 배열 중 뭐가 남았을 지 모르니까 둘다 해주어야 한다.
        while (p1 < n) {
            answer.add(a[p1++]);
        }
        while (p2 < m) {
            answer.add(b[p2++]);
        }
        return answer;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = sc.nextInt();
        }
        int m = sc.nextInt();
        int[] b = new int[m];
        for (int j = 0; j < m; j++) {
            b[j] = sc.nextInt();
        }
        System.out.println(solution(n, m , a, b));
    }

}
```
